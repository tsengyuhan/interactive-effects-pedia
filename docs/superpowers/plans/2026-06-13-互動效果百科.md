# 互動效果百科 實作計畫

> **For agentic workers:** 本計畫依「三 AI 協作協議」執行：隊長（Claude，本對話）逐任務把規格寫入 `team/SPEC.md` 派工給 Codex，Codex 完成後寫 `team/REPORT.md`，隊長以 `git diff` + REPORT 審查。標註【隊長】的任務由隊長親自執行。步驟用 checkbox（`- [ ]`）追蹤。

**Goal:** 建立純靜態、零建置、可離線執行的互動效果展示網站，含三個初始效果與 AI 新增效果的模板/SOP。

**Architecture:** 原生 HTML/CSS/JS；中央登錄檔 `effects/effects.js` 為唯一資料來源；每個效果是自足資料夾，共用 `assets/` 的效果頁外殼（shell）；第三方函式庫全部下載進 `libs/`，禁用 CDN。

**Tech Stack:** Canvas 2D、MediaPipe Tasks Vision（HandLandmarker）、Web Audio API、PowerShell HttpListener（離線本機伺服器）。

**規格來源:** `docs/superpowers/specs/2026-06-13-互動效果百科-design.md`

**測試方式說明:** 本專案無建置工具與測試框架，效果皆為視覺互動，驗證方式為「啟動本機伺服器 → 依每任務的驗證清單逐項目視確認」。每條驗證項目都寫明操作與預期結果。

- [ ] **Step 6: 建立離線本機伺服器 `server.ps1`**

```powershell
# 簡易靜態檔案伺服器——展場離線環境用，零外部相依
# 為什麼需要：攝影機/麥克風權限在 file:// 下不可靠，localhost 是安全環境
$port = 8080
$root = $PSScriptRoot
$mime = @{
  ".html"="text/html; charset=utf-8"; ".js"="text/javascript; charset=utf-8";
  ".mjs"="text/javascript; charset=utf-8"; ".css"="text/css; charset=utf-8";
  ".png"="image/png"; ".jpg"="image/jpeg"; ".svg"="image/svg+xml";
  ".json"="application/json"; ".wasm"="application/wasm";
  ".task"="application/octet-stream"; ".md"="text/plain; charset=utf-8"
}
$listener = New-Object System.Net.HttpListener
$listener.Prefixes.Add("http://localhost:$port/")
$listener.Start()
Start-Process "http://localhost:$port/"
Write-Host "伺服器運行中 http://localhost:$port/ （關閉此視窗即停止）"
while ($listener.IsListening) {
  $ctx = $listener.GetContext()
  try {
    $path = [System.Uri]::UnescapeDataString($ctx.Request.Url.AbsolutePath)
    if ($path.EndsWith("/")) { $path += "index.html" }
    $file = [System.IO.Path]::GetFullPath((Join-Path $root ($path.TrimStart("/") -replace "/", "\")))
    if ($file.StartsWith($root) -and (Test-Path $file -PathType Leaf)) {
      $ext = [System.IO.Path]::GetExtension($file).ToLower()
      $ct = $mime[$ext]; if (-not $ct) { $ct = "application/octet-stream" }
      $bytes = [System.IO.File]::ReadAllBytes($file)
      $ctx.Response.ContentType = $ct
      $ctx.Response.ContentLength64 = $bytes.Length
      $ctx.Response.OutputStream.Write($bytes, 0, $bytes.Length)
    } else { $ctx.Response.StatusCode = 404 }
  } catch { $ctx.Response.StatusCode = 500 }
  $ctx.Response.Close()
}
```

注意 `.wasm` 的 MIME 必須是 `application/wasm`，否則 MediaPipe 無法載入。

- [ ] **Step 2: 實作 `effect.js`**

結構：三層 canvas 疊在 `shell.container` 內——
1. **紙紋層**（最底，只畫一次）：米白底＋兩種程式雜訊：(a) 每像素微量隨機明度抖動（細顆粒）；(b) 數百條極淡的短弧線模擬紙纖維。
2. **墨層**（中，筆畫與暈染都畫在這）：`globalCompositeOperation = "multiply"` 疊在紙上。
3. 紙紋層與墨層分離，**清空按鈕只清墨層**。

筆畫演算法（`pointerdown/move/up`，記得 `setPointerCapture`）：
- 對每段移動，沿線段以 `筆刷大小/4` 間距插值蓋印；每個印是徑向漸層圓（中心 `濃度` 透明度 → 邊緣 0），顏色取參數 `筆墨顏色`。
- 速度（px/ms）越快 → 印的透明度線性遞減、半徑略縮，且隨機略過部分印（飛白）；速度低於門檻 → 在該點登記暈染點。
- 暈染點：`{x, y, r, alpha}` 陣列，rAF 迴圈中每幀 `r += 暈染擴散參數`、`alpha *= 0.92`，以極低透明度畫同色漸層圓，`alpha < 0.005` 時移除。長按不動時每數幀補登記一次，墨會持續積開。

參數（用 `shell.addParam`）：

| key | type | label | 範圍/預設 |
|---|---|---|---|
| density | range | 墨色濃度 | 0.1–1，預設 0.6 |
| bleed | range | 暈染擴散 | 0–1，預設 0.5 |
| size | range | 筆刷大小 | 4–80，預設 24 |
| ink | color | 筆墨顏色 | #1a1a1a |

按鈕：`shell.addButton({label:"清空畫布", onClick})` 清墨層並清空暈染點陣列。

視窗 resize：重設 canvas 尺寸並重畫紙紋（墨跡允許清除，註明於程式註釋）。

- [ ] **Step 3: 驗證**

1. 首頁卡片「水墨筆觸」可進入；分類籤「網頁互動」篩得到。
2. 慢畫 → 邊緣可見逐漸滲開的暈；長按不動 → 墨持續積開成一團。
3. 快速劃過 → 筆畫變淡出現斷續飛白。

6. 雙擊 `index.html` 經 file:// 開啟此效果 → 一切正常（驗收條件 5）。

- [ ] **Step 2: 實作 `effect.js`（classic script，不需 ESM）**

**音訊**：`getUserMedia({audio:true})` → `AudioContext` → `AnalyserNode`（`fftSize: 2048`）。被拒 → `shell.showError("請允許麥克風權限後重新整理頁面；若直接開檔案無法使用，請改用 start.bat 啟動")`。
- 音量：`getFloatTimeDomainData` 算 RMS → `dB = 20 * log10(rms)`，顯示範圍 -60～0 dB。
- 音高：時域自相關——對波形計算各 lag 的正規化相關值，取相關峰值對應 lag，`pitch = sampleRate / lag`；相關峰值 < 0.9 或音量低於門檻時視為「無音高」（顯示「—」）。限制偵測範圍 60–1200 Hz。

**水波模擬**（經典雙緩衝高度場，解析度取畫面的 1/4 以保效能）：

```js
// prev、curr 為 Float32Array(w*h)，每幀：
for (let y = 1; y < h - 1; y++) {
  for (let x = 1; x < w - 1; x++) {
    const i = y * w + x;
    let v = (prev[i-1] + prev[i+1] + prev[i-w] + prev[i+w]) / 2 - curr[i];
    curr[i] = v * damping;   // damping 即衰減率，由音高映射
  }
}
[prev, curr] = [curr, prev];
```

**渲染**：深藍綠水面底色；以高度場 x/y 梯度做明暗（光照感）＋輕微像素位移（折射感），畫到 ImageData 再 `putImageData` 放大至全螢幕（`imageSmoothingEnabled = true`）。

**聲音 → 漣漪映射**：
- 音量超過門檻（約 -45 dB）時投入水滴：在畫面隨機位置（偏中央高斯分布）把高度場壓下，深度與半徑隨音量（-45→0 dB 線性映射）增大；節流每 80 ms 最多一滴。
- 音高 → `damping`：60 Hz → 0.995（低音餘波長）、1200 Hz → 0.96（高音短促），無音高時用中間值 0.985。程式註釋註明：規格允許映射方向對調，以直覺者為準。

**數值顯示**：效果自建左上角覆蓋面板（絕對定位 div，不用 shell 面板）：「音量 −23.4 dB」＋即時水平量條、「音高 440 Hz／—」＋量條，每幀更新。
