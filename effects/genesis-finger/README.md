# 創世紀手指

伸出你的手，與未知之手互動。對著鏡頭伸出手；畫面對面會從你手的「對角對側」伸出一隻手（你的手往上伸，對面的手就從上往下伸），當你的手越往畫面中央移動，對面的手就越靠近，最後兩隻食指指尖相觸，像米開朗基羅《創世紀》那幅畫。指尖相觸停留越久，交會處的火光越亮。手離開畫面，對面的手也會收回。畫面中越多手，就出現越多對面的手（最多依參數設定）。

視覺上，Webcam 畫面會做人像去背、疊在純色背景上，對面的手畫在你的前面。對面的手每次隨機出現五種風格之一（文藝復興壁畫手、卡通手、貓掌、機器人手、外星人手），也可以改用你自己拍下的「神之手」。

右側參數面板可調整手的大小、最多可互動手的數量（1–4），以及「圖片手來源」（內建五種手／我的神之手）。

## 創造神之手

點畫面上方的「✋ 創造神之手」，畫面會切成乾淨的鏡像 webcam 並暫停互動：

1. 把手伸進畫面中央、手指張開，按「拍下」。
2. 系統自動去背、只留下手，並轉正成統一方位（食指朝左）。
3. 預覽時可用筆刷修補——「擦除」清掉沒去乾淨的殘留、「補回」把被誤刪的部分填回來；可調筆畫大小，或「重置」回到剛拍下的樣子。
4. 滿意按「用這隻」，對面伸來的手就全部換成你拍的這隻；反悔按「取消」回到原本的圖片手。

拍好的神之手只保留最新一隻、重整頁面後不保留。隨時可在參數面板把「圖片手來源」切回內建五種。

## 主要技術

- MediaPipe Hands（手部關鍵點）
- MediaPipe Image Segmenter（人像去背）
- getUserMedia
- Canvas 2D（去背合成、神之手筆刷編輯）
- Pointer Events（筆刷修補）

## 需求

- 攝影機
- 建議 Chrome / Edge
- 舊機器 fps 較低
- 需經 `start.bat` 或 HTTPS 開啟（file:// 無法使用）
- 可離線使用（模型與素材都在本地）

## 在自己電腦使用

1. 下載整個專案：
   - `git clone https://github.com/tsengyuhan/interactive-effects-pedia.git`
   - 或在 GitHub 頁面 Download ZIP 後解壓
2. 啟動方式：
   - 雙擊專案根目錄的 `start.bat`
   - 瀏覽器開 `http://localhost:8080/effects/genesis-finger/`
3. 這個效果需要攝影機權限，必須用 `start.bat` 啟動，或透過 HTTPS 開啟。
