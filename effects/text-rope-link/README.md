# 文字繩連連看

用 webcam 偵測畫面中每個人的頭部位置，把輸入文字排成一條條文字繩。單人時文字繩一端黏在鼻子上、另一端自由垂下，隨著頭移動而晃動；多人時每個人各自連到「離自己最近的另一個人」的頭頂，連線受重力在中間自然下垂成弧（就像兩隻手的指尖相連），文字並沿繩持續流動；兩人拉遠超過設定的最遠連接距離，繩子就會斷開、變回各自垂下的單人繩。

## 技術

- MediaPipe Face Detector
- getUserMedia
- Canvas 2D
- Verlet 物理

## 需求

- 攝影機權限
- 現代 Chrome / Edge
- 建議 30 fps 以上
- 本機請用 `start.bat` 啟動伺服器

## 原理

FaceDetector 讀取每個影格的臉部 bounding box，並依鏡像 webcam 座標轉成畫面上的頭頂錨點。系統會替每個人保留短暫追蹤狀態，避免偵測短暫漏失時文字繩立即消失。

繩子用 **時間修正（dt）的 Verlet 物理**（作法取自 guerrillacontra 的 stabilized verlet rope）：速度乘上 dt/prevDt、重力用 a·dt²，所以不同 fps 下手感一致、且更穩定。繩長以多次約束迭代維持剛性（不伸縮），甩動時的彎曲拖尾來自繩身慣性而非鬆軟拉伸。

單人繩只硬釘鼻尖這一端（取自 FaceDetector 的鼻尖關鍵點），其餘節點自由垂掛、隨頭晃動。「文字繩長度（單人）」控制總長。「晃動柔軟度」控制速度阻尼（0.95~0.995）：越高保留越多動量，甩動盪越久、落下反作用越自然。「甩動誇張度」把頭部速度平滑放大後去驅動鼻尖固定端：頭一動固定端被甩出去帶動整條繩，靠慣性產生逐節延遲的彎曲拖尾；頭停下時固定端盪回鼻尖。放大量經低通平滑，避免抖動。

多人時每個人找出離自己最近的另一個人，距離在「最遠連接距離」內就連線。雙人繩兩端各釘一顆頭頂，受重力在中間自然下垂成弧、隨兩人移動晃動（同雙手指尖相連）；文字沿繩持續流動，方向在連線形成時隨機決定、速度由「文字流動速度（雙人）」控制（設 0 即靜止）；超過門檻則拆回各自的單人垂掛繩。

文字會沿著繩子的折線長度取樣，逐字旋轉到切線方向，並可調整文字大小、疏密、粗細、顏色、最遠連接距離、重力、單人繩長度、晃動柔軟度、甩動誇張度與雙人文字流動速度。
