# 文字繩連連看

用 webcam 偵測畫面中每個人的頭部位置，把輸入文字排成一條條受重力影響的繩子。單人時文字繩從頭頂往上甩出再垂下、隨人晃動；多人時每個人各自連到「離自己最近的另一個人」的頭頂，兩人拉遠超過設定的最遠連接距離，繩子就會斷開反彈、變回各自垂下的單人繩。

## 技術

- MediaPipe Face Detector
- getUserMedia
- Canvas 2D
- Verlet 物理

## 需求

- 攝影機權限
- 現代 Chrome / Edge
- 建議 30 fps 以上
- 本機請用 `start.bat` 啟動伺服器

## 原理

FaceDetector 讀取每個影格的臉部 bounding box，並依鏡像 webcam 座標轉成畫面上的頭頂錨點。系統會替每個人保留短暫追蹤狀態，避免偵測短暫漏失時文字繩立即消失。

當只有一個人時，Verlet 節點從頭頂往上初始化、給一點向上初速，再靠重力自然回落下垂。多人時每個人找出離自己最近的另一個人，距離在「最遠連接距離」內就把繩子兩端固定在兩個頭頂錨點，中間節點用距離約束產生鬆弛弧線；超過門檻則拆回各自的單人垂墜繩。

文字會沿著繩子的折線長度取樣，逐字旋轉到切線方向，並可調整文字大小、疏密、粗細、顏色、最遠連接距離與重力。
