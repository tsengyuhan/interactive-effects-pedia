# 文字繩連連看

用 webcam 偵測畫面中每個人的頭部位置，把輸入文字排成一條條文字繩。單人時文字繩一端固定在頭頂，先往上延伸到頭頂上方一個無形的點，再從那裡受重力自然垂下、隨人晃動；多人時每個人各自連到「離自己最近的另一個人」的頭頂，連線會拱成一道弧、文字沿弧持續流動，兩人拉遠超過設定的最遠連接距離，繩子就會斷開、變回各自垂下的單人繩。

## 技術

- MediaPipe Face Detector
- getUserMedia
- Canvas 2D
- Verlet 物理

## 需求

- 攝影機權限
- 現代 Chrome / Edge
- 建議 30 fps 以上
- 本機請用 `start.bat` 啟動伺服器

## 原理

FaceDetector 讀取每個影格的臉部 bounding box，並依鏡像 webcam 座標轉成畫面上的頭頂錨點。系統會替每個人保留短暫追蹤狀態，避免偵測短暫漏失時文字繩立即消失。

單人繩用 Verlet 物理：只有頭頂那一端硬釘，繩上的錨點節點每幀被「軟拉」往頭頂上方的無形點（不硬釘），所以整條繩維持柔軟、會隨頭部移動晃動，重力模擬沿用「玩弄文字於指尖」效果。「文字繩長度（單人）」控制總長、「無形錨點高度（單人）」控制往上延伸的距離。

多人時每個人找出離自己最近的另一個人，距離在「最遠連接距離」內就連線。雙人連線不受重力影響，而是一道往上拱起的固定弧線（二次貝茲），「拱起高度（雙人）」控制弧度；連上之後文字會沿弧持續流動，方向在連上當下隨機決定、速度由「文字流動速度（雙人）」控制；超過門檻則拆回各自的單人垂墜繩。

文字會沿著繩子的折線長度取樣，逐字旋轉到切線方向，並可調整文字大小、疏密、粗細、顏色、最遠連接距離、重力、單人繩長度、無形錨點高度、拱起高度與文字流動速度。
