# 文字繩連連看

用 webcam 偵測畫面中每個人的頭部位置，把輸入文字排成一條條文字繩。單人時文字繩一端黏在鼻子上、另一端自由垂下，隨著頭移動而晃動；多人時每個人各自連到「離自己最近的另一個人」的頭頂，連線受重力在中間自然下垂成弧（就像兩隻手的指尖相連），兩人拉遠超過設定的最遠連接距離，繩子就會斷開、變回各自垂下的單人繩。

## 技術

- MediaPipe Face Detector
- getUserMedia
- Canvas 2D
- Verlet 物理

## 需求

- 攝影機權限
- 現代 Chrome / Edge
- 建議 30 fps 以上
- 本機請用 `start.bat` 啟動伺服器

## 原理

FaceDetector 讀取每個影格的臉部 bounding box，並依鏡像 webcam 座標轉成畫面上的頭頂錨點。系統會替每個人保留短暫追蹤狀態，避免偵測短暫漏失時文字繩立即消失。

繩子用 Verlet 物理，作法與「玩弄文字於指尖」一致。單人繩只硬釘鼻尖這一端（取自 FaceDetector 的鼻尖關鍵點），其餘節點完全自由垂掛，所以整條繩柔軟、字與字之間是自然的鏈狀彎曲而非僵直，並隨頭移動晃動。「文字繩長度（單人）」控制這條繩的總長。「晃動柔軟度」控制速度阻尼：越高保留越多動量，快速甩頭時文字繩會被甩起、落下時帶自然的反作用慣性。頭部錨點帶速度預測與短暫追蹤，移動時較不會慢半拍或抖動。

多人時每個人找出離自己最近的另一個人，距離在「最遠連接距離」內就連線。雙人繩兩端各釘一顆頭頂，受重力在中間自然下垂成弧、隨兩人移動晃動（同雙手指尖相連）；超過門檻則拆回各自的單人垂掛繩。

文字會沿著繩子的折線長度取樣，逐字旋轉到切線方向，並可調整文字大小、疏密、粗細、顏色、最遠連接距離、重力、單人繩長度與晃動柔軟度。
